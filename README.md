## Getting Started

### Prerequisites

- C++ compiler (e.g., GCC, Clang)
- OpenGL library
- GLFW library
- GLEW library

### Installation

1. Clone the repository:

```bash
git clone https://github.com/TheSmileyDroid/SCraft.git
```

2. Build the project:

```bash
cd SCraft/build
cmake ..
ninja
```

3. Run the game:

```bash
./SCraft
```
